## Level 2 쿼드압축 후 개수 세기

모두 1이나 0이 아니라면 배열을 4분할 하여서 재귀함수에 넣어야 한다.\

재귀 부분을 어떻게 해야할지 몰라서 검색으로 좀 찾아보고 풀었다.\
