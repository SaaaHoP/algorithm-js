# 2021.09.13 알고리즘 풀이

## Level 2 점프와 순간 이동

문제 자체가 어려웠던건 아니지만, 생각을 하는데 좀 오래걸렸다.\
최대의 효율로 움직이려면 어떤 방법이 효과가 좋을까 고민을 오래했다.\
마지막으로 든 생각이, 주어진 숫자가 짝수면 2로 나누고, 홀수면 1을 빼서 짝수로 만든 뒤,\
2로 나누고 카운트를 세주었다. 이렇게 숫자가 0이 될때까지 while문을 돌리면, 정답이 나왔다.
